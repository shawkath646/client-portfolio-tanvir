## 🚀 Getting Started

### Prerequisites

- **Web Browser** - Any modern browser (Chrome, Firefox, Safari, Edge)
- **Git** - For cloning and deploying to GitHub Pages

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/shawkath646/client-portfolio-tanvir.git
   cd client-portfolio-tanvir
   ```

2. **Open the project**
   - Double-click `index.html` to open in your default browser
   - Or use a local server for better development experience

3. **Using Live Server (VS Code)**
   ```bash
   # Install Live Server extension in VS Code
   # Right-click index.html and select "Open with Live Server"
   ```

4. **Using Python HTTP Server**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Open http://localhost:8000 in your browser
   ```

### 🌐 GitHub Pages Deployment

This project is optimized for GitHub Pages. To deploy:

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Deploy portfolio"
   git push origin main
   ```

2. **Enable GitHub Pages**
   - Go to repository Settings
   - Navigate to "Pages" section
   - Under "Source", select branch: `main` and folder: `/ (root)`
   - Click "Save"

3. **Access Your Site**
   - Your site will be live at: `https://USERNAME.github.io/REPOSITORY-NAME/`
   - Example: `https://shawkath646.github.io/client-portfolio-tanvir/`

4. **Custom Domain (Optional)**
   - If you have a custom domain, rename `CNAME.example` to `CNAME`
   - Edit `CNAME` and add your domain name
   - Configure DNS settings with your domain provider

**Note**: The `.nojekyll` file is included to prevent Jekyll processing. All paths are relative for GitHub Pages compatibility.

### Customization

1. **Update Personal Information**
   - Edit `index.html` to change name, skills, and contact details
   - Replace social media links with actual profiles

2. **Add/Replace Images**
   - Place your images in the `images/` folder
   - Update image paths in `index.html`
   - Recommended image formats: JPG, PNG, WebP

3. **Modify Styles**
   - Edit `style.css` to customize colors, fonts, and layouts
   - Adjust responsive breakpoints as needed

4. **Update Typewriter Text**
   - Modify the `data-type` attribute in `index.html` hero section
   - Add or remove items from the array
